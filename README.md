# Dhruvi Khatrani — Portfolio (Next.js)

Premium portfolio converted from the Stitch HTML design to **Next.js 16**, **Tailwind CSS v4**, and **GSAP** animations.

## Features

- Dark “Obsidian Flow” design system
- Mobile-responsive layout with hamburger navigation
- GSAP hero and scroll animations
- Section components: Hero, About, Skills, Experience, Projects, Contact

## Getting started

```bash
cd portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project structure

- `src/app/` — layout, global styles, home page
- `src/components/` — UI sections
- `src/lib/data.ts` — content (skills, projects, links)

## Customize

Edit `src/lib/data.ts` for projects, experience, and social links.
